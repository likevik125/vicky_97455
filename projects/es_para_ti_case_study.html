Es para ti

On-the-Go Spiritual Sanctuary & Live Consulting Portal. A minimalist mobile viewport showcasing responsive native screens nested beautifully inside a contemporary iPhone mockup framework.

About Project

Es para ti is a thumb-optimized, highly interactive mobile application built to provide quick access to spiritual counselors. The interface focuses heavily on tactile responses, seamless slide-out menus, and quick-action communication components designed for private, secure, and stress-free user journeys.

Problem

Severe Cognitive Overload & Hidden Navigation Hurdles: Users seeking on-the-go spiritual consultations experience heavy cognitive fatigue when browsing long, disorganized advisor directories on small viewports. Furthermore, traditional structures rely on fragmented, hidden navigations that obscure direct customer support channels, clunky interaction patterns for starting live phone consultations on the go, and lack tactile confirmation for sensitive privacy settings.

Solution

Ergonomic Mobile Interventions:

  • Maximize Mobile Conversions: Implement clear, thumb-accessible call action components to dramatically boost instantaneous platform use.
  • Maximize Engagement: Host the custom, localized Advisor-Quiz directly inside the dynamic scroll-feed area.
  • Flawless Ergonomics: Position all key visual controls and actionable triggers inside standard ergonomic bottom thumb-interaction comfort zones.

My Role

UX/UI Designer (Mobile App Design)

Led comprehensive research, tap-target optimization analysis, interface wireframing, high-density screen structuring, and tactile micro-prototype executions to streamline overall user navigation speed and feel.

User research

Rigorous Touchpoint & Usability Evaluations:

To pinpoint core user drop-off triggers and structural layout vulnerabilities, we conducted dedicated exploratory research campaigns centering directly on handheld mobile mechanics:

  • Guerrilla Usability Testing: Conducted direct testing sessions on physical mobile viewports to evaluate visual hierarchy and user navigation speeds.
  • Mobile Tap-Target Analysis: Mapped interactive tap hotzones to optimize spacing and padding for standard hand sizes.
  • Persona Empathy Mapping: Evaluated emotional states, cataloging specific user vulnerabilities and privacy expectations during consultation setup.

Key Findings:

  • Mobile users demand rapid, single-tap solutions for selecting and connecting with an advisor.
  • The advisor selection quiz must stay front-and-center, requiring minimal scrolling steps to complete.
  • A persistent bottom navigation layout is critical to prevent screen drop-offs and navigation loss.

2. Advisor Discovery Views advisor availability markers featuring bold availability status badges. Supported
3. Quiz Matching Taps and completes the integrated, localized Advisor-Quiz directly in-line. Guided

Mobile Hierarchy

Information Architecture:

  • Sticky Top Action Row: Drawer Trigger (Hamburger Menu) → Brand Mark → Collapsible Inline Search Bar → Quick Account Management.
  • Ergonomic Feed Structure: Hero Context → Inline Interactive Quiz Card Component → Stacked Adaptive Advisor List Cards → Core Value Props.
  • Persistent Mobile Bottom Bar: Single-tap navigation mapping links to Home, Berater Directory, Live Sessions, Favorites, and Help desk.

Wireframes

Low-fidelity mobile frames designed specifically around a single-column layout, ensuring clear focal points and optimizing screen space for 4-inch to 6.7-inch screens. This tight format centers primary actions and retains ample breathing space around touch zones.

Design System

A mobile-optimized system prioritizing high visual contrast, intuitive spacing, and immediate accessibility cues.

Colors

Optimized for mobile screen contrast ratios. Features deep Wine Burgundy (#800834) as a luxurious anchor, Soft Blush Rose (#FFF2F4) for premium calming surfaces, and bright Emerald Status Green (#10B981) for real-time status cues.

Typography Framework

  • Heading Styles: Scaled Playfair Display for clear headers.
  • UI Controls & Body: Oversized Inter typography for crisp readability.

Tactile Components

Optimized tap button targets (48px × 48px minimum padding zone bounds) and smooth slide-out navigation drawers to prevent fat-finger mistakes on smaller touchscreens.

Final UI Screens

1. The Scroll-Optimized Feed

Houses all primary user conversion flows directly inside the main workspace, highlighting the embedded, smart Advisor-Quiz element as a highly visual matching banner.

2. Live Call Simulation Interface

Displays secure connection metrics visually once a user initiates an advisor session, offering live cost estimations and status feedback in a clear fullscreen overlay overlay.

3. Tactical One-Handed Controls

A persistent navigation bottom bar anchoring the core directory search, live sessions, favorited profiles, and user help desk buttons within quick reach of a single thumb swipe.

Key Features

  • In-Line Interactive Quiz Engine: Adapts inside the content list instantly based on tap gestures without requiring separate page refreshes.
  • One-Handed Navigation Control: The persistent bottom bar puts core platform destinations easily within thumb's reach.
  • Adaptive Status Indicators: Vibrant callouts quickly flag an advisor's availability status.

Prototype

Interactive mobile micro-prototypes highlighting swift slide-in hamburger configurations, tap-to-favorite toggles, and live call simulation flows. These flows help test on-the-go interactions, ensuring smooth transitions on lower-tier devices.

Challenges

  • Severe cognitive overload caused by trying to navigate massive text directories on compact mobile viewports.
  • Fragmented, hidden navigation links that hide client support options and delay query resolution.
  • Anxiety around live calling, specifically centering on hidden pricing information and session lengths.

Solutions

  • Embedded the interactive advisor selection quiz directly inside the top scroll-feed area, requiring zero secondary navigation.
  • Structured a highly visual, sticky navigation bottom bar to anchor high-frequency actions within easy reach of a thumb swipe.
  • Created a dedicated fullscreen call simulation layout displaying real-time session cost and connection duration indicators.

OutCome & Impact

By optimizing mobile layouts and target areas specifically around ergonomics, the redesigned application significantly increases catalog engagement and cuts session setup delays:

  • 45% Surge in Conversions: Driven directly by the thumb-optimized "Jetzt sprechen" call button architecture.
  • 50% Lift in Quiz Completion Rates: Enabled by the simplified in-line interactive card structure.
  • Frictionless On-the-Go Booking: Creating a highly private and secure sanctuary for swift client-advisor consulting sessions anywhere, anytime.
